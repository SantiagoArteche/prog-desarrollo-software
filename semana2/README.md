# Ejercicio 38

Farenheit (F) con la fórmula:
F = 9/5 C + 32
De Farenheit a Centígrados con la fórmula:
C = (F – 32) 5/9
Elaborar un programa que lea una temperatura en alguna de las dos escalas y
obtenga e imprima la temperatura en su equivalente.
